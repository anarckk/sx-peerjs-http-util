我希望服务端的使用方式可以注册和销毁处理器：

```js
import { PeerJsWrapper } from 'sx-peerjs-http-util';

const wrapper = new PeerJsWrapper();

// 注册处理器
wrapper.registerHandler('/api/hello', function () {
    return {
        message: 'hello'
    };
});

// 销毁处理器
wrapper.unregisterHandler('/api/hello');
```

在处理器中可以直接返回数据，wrapper 里面，要自动对其进行装箱。
同理，send函数的返回Promise的时候，要自动进行拆箱。

---

PeerJsWrapper 的发送函数，在 `conn.on('data'` 里，获得响应之后，拆包之前，要校验返回状态是否正确
