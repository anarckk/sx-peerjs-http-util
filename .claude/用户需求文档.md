# 用户需求文档

## 项目概述

编写一个库工具，把浏览器端的 PeerJS 封装成简单易用的类似 HTTP 的 API。

## 核心功能

### API 封装

封装 API，底层使用 PeerJS。

### 入参与返回值

- 入参：对端设备的 peerId 和传递的 request 参数
- 返回值：Promise<Response>，即对端设备的返回值

### 连接管理

每次请求都重新连接 conn。

### 请求/响应简化

request、response 不需要 header，可以简化。

## 目标环境

这个库不是给 Node.js 后端环境设计的，而是为浏览器设计的，未来仅提供给浏览器端使用。

## 技术要求

使用 TypeScript。

## 测试要求

要有 E2E 测试。

### 测试范围

E2E 测试只需要测试 Chrome 浏览器即可。
