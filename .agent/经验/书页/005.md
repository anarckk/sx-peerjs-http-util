# Playwright测试不稳定 - 005

## 问题
`Execution context was destroyed`

## 原因
`Promise.all` 并发执行 page 操作导致竞态

## 解决
串行执行：
```typescript
// 错误
await Promise.all([pageA.goto(...), pageB.goto(...)]);
const [idA, idB] = await Promise.all([...]);

// 正确
await pageA.goto(...);
await pageB.goto(...);
const idA = await pageA.evaluate(...);
const idB = await pageB.evaluate(...);
```
