# Vue3 单文件打包经验

## 问题
Vue3 项目打包后生成多个文件（HTML、JS、CSS），不方便分发和部署。

## 解决方案
使用 `vite-plugin-singlefile` 插件，将所有资源内联到单个 HTML 文件中。

## 实现步骤

### 1. 安装插件
```bash
npm install --save-dev vite-plugin-singlefile
```

### 2. 配置 vite.config.js
```js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import { viteSingleFile } from 'vite-plugin-singlefile'

export default defineConfig({
  plugins: [vue(), viteSingleFile()],
  build: {
    target: 'esnext'
  }
})
```

### 3. 打包
```bash
npm run build
```

## 注意事项

1. **导入方式**：使用 `{ viteSingleFile }` 而非默认导入 `viteSingleFile`
   - ❌ `import viteSingleFile from 'vite-plugin-singlefile'`
   - ✅ `import { viteSingleFile } from 'vite-plugin-singlefile'`

2. **build.target**：设置为 `'esnext'` 以支持最新 JavaScript 特性

3. **文件大小**：打包后文件较大（500KB+），因为包含所有依赖

4. **浏览器兼容**：可直接用浏览器打开，无需服务器

## 优点
- 分发方便：单个 HTML 文件即可运行
- 部署简单：无需配置静态资源路径
- 离线可用：所有资源已内联

## 缺点
- 文件体积大
- 首次加载慢
- 无法利用浏览器缓存（CSS/JS 分离时可以）
