# `经验`规则

经验分为目录(`.agent/经验/目录.md`)和书页(`.agent/经验/书页/{序号}.md`)

`目录.md`格式:
```
分类/
├── 经验标题 - {序号，比如001}
└── 分类/
    ├── 经验标题 - 002
    └── 经验标题 - 003
```

经验负责存放：
1. 犯过的错误和处理的方法。
2. 工作中学习到的新知识。
3. 产品需求设计、技术方法、或其他有未来参考价值但不值得放入长期记忆的知识

经验可以通过目录进行索引寻找，通过序号(001、002、003...)找到对应文件
