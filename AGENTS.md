# e2e文档规范

在项目目录 AGENTS.md 中维护一个篇章 `# e2e测试用例`，格式如下：

- [x] {测试项A}
- [ ] {测试项A}

记录所有的e2e测试项，以及其是否通过e2e测试。

---

**运行方式**：
1. 启动私有信令服务器：`cd peerjs-server && node server.js`
2. 启动 HTTP 服务器：`npx serve -l 8080`
3. 运行测试：`npm run test:e2e`
