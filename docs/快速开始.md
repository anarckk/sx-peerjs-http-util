# 快速开始

本工具将 PeerJS 封装成类似 HTTP 的 API，让浏览器间的 P2P 通信像调用 HTTP 接口一样简单。

## 安装

**NPM:**
```bash
npm install sx-peerjs-http-util peerjs
```

**CDN:**
```html
<script src="https://unpkg.com/sx-peerjs-http-util/dist/index.umd.js"></script>
```

## 基础通信

### 1. 创建实例

```typescript
import { PeerJsWrapper } from 'sx-peerjs-http-util';

const wrapper = await PeerJsWrapper.create();
console.log('我的 Peer ID:', wrapper.getPeerId());
```

### 2. 接收请求（注册处理器）

```typescript
// 注册处理器，类似 HTTP 路由
wrapper.registerHandler('/api/hello', async (from, data) => {
  console.log('收到来自', from, '的消息:', data);
  // 可以异步处理，返回 Promise
  await new Promise(r => setTimeout(r, 100));
  return { message: '你好，我是 B' };
});
```

### 3. 发送请求

```typescript
// 发送请求，类似 fetch
const response = await wrapper.send('对方PeerID', '/api/hello', { name: '你好' });
console.log('收到回复:', response);
```

## 语音/视频通话

### 发起通话

```typescript
// 语音通话
const call = await wrapper.call('对方PeerID');

// 视频通话
const videoCall = await wrapper.call('对方PeerID', { video: true });
```

### 接听来电

```typescript
wrapper.onIncomingCall(async (event) => {
  // 显示来电提示 UI
  const session = await event.answer();  // 接听
  // 或 event.reject();  // 拒绝
});
```

## 中继通信

当两端无法直连时（NAT 穿透问题），库会自动通过中继节点转发。

`send()` 方法会自动尝试：
1. 直连目标节点
2. 失败则通过路由表中已知的中继节点转发
3. 路由表为空时广播路由发现请求

## 调试模式

```typescript
const wrapper = new PeerJsWrapper('自定义ID', true);
```

开启后会在控制台输出详细日志。
